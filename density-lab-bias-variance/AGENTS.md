# AGENTS.md

This repository is an educational Python project about density estimation,
Gaussian maximum likelihood estimation, KDE, GMM, bias-variance tradeoff, and
the curse of dimensionality.

Guidelines for future coding agents:

- Keep code beginner-friendly and heavily commented.
- Prefer clear NumPy, scipy, scikit-learn, and matplotlib code over clever abstractions.
- Do not use seaborn; plots should use matplotlib only.
- Keep random seeds controllable through `random_state`.
- Do not hardcode absolute paths.
- Make generated figures and reports reproducible from `python -m density_lab.experiments`.
- Preserve Vietnamese explanations in README, notebook, report, and Streamlit app.
